# **App Name**: Sportlight

## Core Features:

- Player Profile Creation: Allow players to create profiles with details like personal information, achievements, and skill videos.
- Club Registration: Enable clubs to register and create profiles specifying their criteria for player selection and verification documents.
- AI Video Guidance: Offer personalized video recording guidance to players using OpenCV and Mediapipe for real-time landmark detection to ensure proper camera alignment and visibility.
- Sport-Specific Criteria: Implement specific recording criteria for various sports to ensure the quality and relevance of audition videos.
- AI-Powered Chatbot: Integrate an AI chatbot to handle basic to moderate player and club queries, providing instant support and information. Use an LLM as a tool.
- Profile Filtering and Sorting: Implement efficient filtering and sorting algorithms to allow clubs to easily find player profiles based on sport, skill, and verified documents.
- Notifications: Notify shortlisted players and clubs of relevant updates and achievements with automated congratulatory certificates.
- Player Dashboard: Dashboard where all the player profiles will be displayed and sorted on the basis of the sports they are choosing.
- AI Video Checker: AI-powered video checker which will check all the angles and the timing of the video which is given by the user, ensuring each criteria of the sports is fulfilled. Use an LLM as a tool.

## Style Guidelines:

- Primary color: Glittering green (#7CFC00) for a vibrant and energetic feel, symbolizing growth and opportunity.
- Background color: Black (#000000) to create a modern and sophisticated look.
- Accent color: Purple tincture (#E0B0FF) to add a touch of uniqueness and contrast.
- Headline font: 'Poppins' (sans-serif) for a contemporary, geometric, fashionable, avant-garde look. Body font: 'PT Sans' (sans-serif) to combine a modern look and a little warmth or personality.
- Use vibrant and dynamic icons related to sports to enhance user engagement.
- Ensure a clean, intuitive layout optimized for both mobile and web platforms.
- Incorporate subtle, engaging animations to highlight key interactions and achievements.
